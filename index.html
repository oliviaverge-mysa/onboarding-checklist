SolidWorks Training
Get comfortable with the primary CAD tool
0 / 4
Watch the SolidWorks beginner tutorial
If you're new or rusty, start here. Get comfortable with creating parts, assemblies, and drawings before diving into Mysa product files.
Beginner Tutorial
Understand the file types used at Mysa
SLDPRT is your main working file. STL is for 3D printing and JLC orders. STEP is for exporting to Arena for suppliers. Know when to use each.
Review tolerancing and fit guidelines
Standard mating parts get 0.125mm clearance per side (0.25mm total). Critical fit parts use 0.25mm on one side as a safety buffer for printer inaccuracies and material expansion.
Build a practice model and export it
Design something simple from scratch — a bracket, enclosure, or organizer. Export as STL and save your SLDPRT to your work term folder. This is your first real deliverable!

3D Printing
Learn both the FDM (Prusa) and SLA (resin) printers
0 / 6
Read the Prusa XL printer setup guide
Learn how to check the print bed, load/unload filament, and prepare the printer before starting a job. The print bed is magnetic — don't forget to check it's seated properly.
Prusa XL Help
Slice your first model in PrusaSlicer
Import your practice STL. Set PLA settings: bed 50–60°C, nozzle 215°C, layer height ≤0.2mm. Orient the model on its flattest face to minimize supports. Slice and review the preview.
Complete your first Prusa print
Save your GCode to the Prusa USB, insert it into the printer, and select your file. Stay and watch the first 1–2 layers to make sure adhesion looks good before walking away.
Read the resin (Form 3 SLA) safety requirements
Resin is a sensitizer. Before touching anything: nitrile gloves (in the toolbox drawer), safety glasses, and lab coat are mandatory. Tie back long hair, remove dangling jewelry.
Learn the PreForm software workflow
Open your STL in PreForm. Select "GroovyLark" as the printer. Use the Magic Wand (one-click auto-orient and support generation). Upload to the printer via Wi-Fi.
PreForm Download
Learn the wash and cure process
After printing: scrape excess resin back into the tank, wash in Form Wash (10–15 min IPA), let fully dry, remove supports with flush cutters, then cure in Form Cure. Never mix resin types.

Arena Training
Product lifecycle management and BOMs
0 / 5
Watch the Arena video tutorials
In Arena, find the Help section (the ? circle icon). Watch: Getting Started → Searching in Arena → Items → BOMs → Files → Suppliers and Sharing. Take your time with these.
Arena Video Tutorials
Understand Mysa's part number system
900- = Finished Goods. 800- = Assemblies (850- packaging, 810- product, 820- subassembly). 472- = screws/bolts, 450- = nuts. Each part category has its own PN prefix.
Build a practice BOM in the Sandbox
Make sure you're in the green "MYSA Sandbox" Arena environment. Go to bricklink.com, pick a simple 15–20 piece Lego set, and document it as a practice product assembly with a full BOM.
BrickLink
Learn how to submit an ECO
ECOs (Engineering Change Orders) inform collaborators of changes and request approval. Practice the flow: update the BOM → navigate to Changes → Create → Engineering Change Order → add affected items → submit. Always check "Share with Suppliers".
Practice the Slack ECO notification flow
After submitting an ECO, copy the category → description info from Arena and paste it as a code block in the appropriate Slack channel. Tag the relevant managers. This is standard process for every ECO.
